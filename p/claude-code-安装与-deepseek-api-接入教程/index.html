Claude Code 安装与 DeepSeek API 接入教程

Windows 平台下 Claude Code 的完整安装教程，包含 DeepSeek API 接入与 cc-switch 配置

28286-28-287
阅读时长: 3 分钟

前言

Claude Code 是 Anthropic 推出的命令行 AI 编程助手，配合 DeepSeek V4 模型可以大幅降低使用成本。本教程记录 Windows 平台下从零安装到成功使用的全过程。

一、环境准备

1.1 所需软件

软件用途下载地址
Node.jsClaude Code 运行环境nodejs.org → 点击 Releases 下载 Windows 版本
cc-switchAPI 代理切换工具，用于接入 DeepSeekGitHub 搜索 cc-switch，下载 Windows 版本
Git版本管理（Claude Code 依赖）git-scm.com

1.2 配置 npm 国内镜像

由于 npm 默认源在国内访问较慢，先切换到国内镜像：

1

npm config set registry https://registry.npmmirror.com/

1.3 验证安装

安装完上述软件后，在终端中分别执行以下命令，确认均有版本号输出：

1
2
3

node -v      # 应输出 v20.x 或更高
npm -v       # 应输出 10.x 或更高
git -v       # 应输出 git version 2.x

二、安装 Claude Code

2.1 全局安装

打开终端（CMD 或 PowerShell），执行：

1

npm install -g @anthropic-ai/claude-code

2.2 验证安装

1

claude --version

有版本号输出即表示安装成功。

2.3 跳过登录验证

Claude Code 默认要求登录 Anthropic 账号。我们通过 DeepSeek API 接入，不需要登录，直接修改配置文件跳过：

  1. 打开文件资源管理器，在地址栏输入 %USERPROFILE% 回车
  2. 找到 .claude 文件夹（注意：是 .claude 文件夹，不是 claude.json）
  3. 进入 .claude 文件夹，找到 claude.json 文件
  4. 在文件末尾添加 "hasCompletedOnboarding": true

关键修正：很多教程误写成"claude.json 文件夹"，实际上是用户目录下的 .claude 文件夹，里面有 claude.json 配置文件。

修改后的 claude.json 示例：

1
2
3

{
  "hasCompletedOnboarding": true
}

三、接入 DeepSeek API

3.1 注册并获取 API Key

  1. 访问 DeepSeek 开放平台 注册账号。
  2. 进入 API Keys 页面，点击 创建 API Key
  3. 为 Key 命名（如 claude-code），创建后立即复制保存——关闭弹窗后将无法再次查看

确保账户内有一定余额（最低充值通常 ¥10 即可使用很久）。

3.2 配置 cc-switch

cc-switch 是一个开源的 API 代理工具，负责将 Claude Code 的 Anthropic API 请求转发到 DeepSeek 的 API 端点。

  1. 下载并启动 cc-switch（Windows 版本）
  2. 点击 添加 按钮
  3. 选择 DeepSeek 作为后端
  4. 填入之前复制的 API Key
  5. 将所有模型名称修改为：deepseek-v4-pro[1m]

为什么用 cc-switch？ Claude Code 默认调用 Anthropic 官方 API，费用较高。cc-switch 在本地拦截请求并转发到 DeepSeek，让你用 DeepSeek 的价格享受 Claude Code 的交互体验。

四、验证使用

  1. 打开终端，输入 claude 启动
  2. 首次启动会要求选择工作目录，选择你常用的代码目录即可
  3. 进入对话后，询问：“你是什么模型？”
  4. 若回答为 deepseek-v4-pro[1m]，说明接入成功

4

# 启动 Claude Code
claude

# 进入后在对话中输入
你是什么模型？

五、常见问题

问题可能原因解决方案
claude 命令不存在npm 全局安装路径未加入 PATH将 %APPDATA%\npm 加入系统环境变量
cc-switch 连接失败API Key 无效或余额不足检查 DeepSeek 平台的 API Key 状态和余额
模型回答不是 DeepSeekcc-switch 未正确拦截检查 cc-switch 是否在运行，模型名是否配置正确
安装卡在 npm installnpm 源连接慢确认已切换国内镜像源

总结

整个流程概括为四步：

  1. 装环境 — Node.js + Git + cc-switch
  2. 装 Claude Code — npm install -g + 跳过登录
  3. 接 API — DeepSeek 注册 + cc-switch 配置
  4. 验证 — 启动对话，确认模型正确

配置完成后即可在终端内享受 AI 辅助编程，使用体验与官方 Claude Code 一致，但成本仅为 DeepSeek API 的调用费用。
